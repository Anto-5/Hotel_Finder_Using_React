# Hotel Finder

Hotel Finder is a React application that allows users to browse hotels using data from a REST API. Users can search hotels, filter them by city, sort them by price, and view detailed information about each hotel.

## Features

- Browse hotel listings
- Search by name or location
- Filter by city
- Sort by price
- View hotel details and image gallery
- Responsive design

## Technologies Used

- React
- Vite
- React Router DOM
- CSS
- REST API

## Getting Started

```bash
npm install
npm run dev
```
