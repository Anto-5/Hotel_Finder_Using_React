# Hotel Finder

A responsive hotel search application built with React and Vite that integrates with a REST API to display hotel listings and detailed hotel information.

## Features

- View hotel listings
- Search hotels by name or location
- Filter hotels by city
- Sort hotels by price
- View detailed hotel information
- Image gallery for each hotel
- Responsive design
- Loading spinner and empty state handling

## Technologies Used

- React
- Vite
- React Router DOM
- CSS3
- REST API

## API

https://demohotelsapi.pythonanywhere.com/

## Getting Started

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
